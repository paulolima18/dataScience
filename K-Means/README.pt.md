# Projetos K-Means

K-Means é um método de não supervisionado de Machine Learning.
K-Means clustering é utilizado para identificar padrões que não tenham sido documentados explicitamente previamente.
Utilizações:
  - Deteção de fraude
  - Compreensão de imagens
  - Motores de busca
